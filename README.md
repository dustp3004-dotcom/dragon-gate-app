# 🐉 Dragon Gate Asian Pantry — PWA Setup Guide

## How to go LIVE on Vercel (Free) in 5 minutes

### Step 1 — Install Node.js
Go to https://nodejs.org and download the LTS version. Install it.

### Step 2 — Install Git
Go to https://git-scm.com and install it.

### Step 3 — Create a GitHub account
Go to https://github.com and sign up (free).

### Step 4 — Upload your project to GitHub
1. Go to github.com and click "New repository"
2. Name it: dragon-gate-app
3. Click "Create repository"
4. Upload all these files to the repository

### Step 5 — Deploy to Vercel
1. Go to https://vercel.com
2. Sign up with your GitHub account
3. Click "New Project"
4. Select your dragon-gate-app repository
5. Click "Deploy"
6. Wait 2 minutes...
7. ✅ Your app is LIVE!

### Your app URL will be:
https://dragon-gate-app.vercel.app

### How customers install it as an app:
1. Send them the Vercel link on WhatsApp
2. They open it in Chrome on their phone
3. Chrome shows "Add to Home Screen" banner
4. They tap it — done! App icon appears on their phone 🎉
